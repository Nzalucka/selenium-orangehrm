# selenium-orangehrm

Hybrid UI + API Test Automation Framework for [OrangeHRM](https://opensource-demo.orangehrmlive.com)
## About
Hybrid framework — REST Assured creates/deletes test data via API, Selenium tests UI. Page Object Model pattern.

## Stack
Java · Selenium WebDriver 4 · REST Assured · TestNG · Allure · WebDriverManager · AssertJ · Maven
